# Описание класса Client
Класс для работы с клиентом

## Атрибуты

* Comment : string
* Language : List<Language>
* Sex : bool
* TypeOfOccup : string

## Описание атрибутов

* Comment : string - комментарий к клиенту
* Language : List<Language> - список изучаемых языков
* Sex : bool - пол
* TypeOfOccup : string - тип обучения
